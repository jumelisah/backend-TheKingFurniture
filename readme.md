## üìå How To Install?

- Clone This Repo
```
git clone https://github.com/eexxprd/backend-TheKingFurniture
```
- Go To Folder Repo
```
cd backend-TheKingFurniture
```
- Install Module
```
npm install
```
- Type ``` npm run dev ``` To Start The Project
- Set file ```.env```
